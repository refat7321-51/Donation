# 🇵🇸 Gaza Emergency Relief Campaign

A state-of-the-art, high-end, responsive single-page web application designed to collect emergency humanitarian donations for families in Gaza. This website features a premium modern aesthetic, utilizing elegant layouts, harmonious branding tones, fluid hover micro-animations, and full responsive support.

Live Preview: [https://refat7321-51.github.io/Donation/](https://refat7321-51.github.io/Donation/)

---

## ✨ Premium Features

*   **🏆 State-of-the-Art Aesthetic:** High-contrast professional design featuring a brand identity centered around Crimson Red (representing urgency) and Emerald Green (representing hope and growth).
*   **🫧 Sticky Glassmorphism Header:** A beautifully styled navigation bar with dynamic backdrop blurs (`backdrop-filter`) and smooth color transitions on user scroll.
*   **📱 Seamless Responsiveness:** Carefully tailored CSS media query breakpoints ensuring perfect display on all modern screens (Ultra-Wide, Desktop, Tablet, and Mobile).
*   **🍔 Mobile Hamburger Menu:** A stylish custom navigation sidebar that collapses and transitions smoothly on smaller devices.
*   **❤️ Interactive Preset Amounts:** An interactive grid of quick-select donation tiles (`100tk`, `200tk`, `500tk`, `1000tk`, `5000tk`) with high-fidelity active border overlays.
*   **⌨️ Custom Input Focusing:** Dynamic javascript input handling that seamlessly manages selected presets when custom manual entry is activated.
*   **🍞 High-Quality Vector Icons:** Replaced basic emojis with clean, crisp, local vector assets (`food`, `medicine`, `shelter`, `blood`) representing how donations are used.
*   **🖼️ Professional Hover Story Gallery:** A modern photo-story grid. Hovering over a photo smoothly triggers a glassmorphic description card detailing local stories with photo credits.
*   **🔔 Interactive Success Toast Alerts:** Employs vanilla JavaScript success popups when users subscribe to the updates newsletter or process a simulated donation.
*   **🚀 Back-to-Top Button:** A smooth scrolling floating trigger that dynamically appears when the user scrolls past the top fold.

---

## 🛠️ Technology Stack

*   **Markup:** HTML5 (semantic, structured elements, accessibility-friendly)
*   **Styling:** CSS3 (modern flexbox, grid, custom properties/variables, media queries, transition animations)
*   **Interactions:** Vanilla JavaScript (ES6+, DOM Manipulation, dynamic classes, and scroll event listeners)
*   **Icons:** Font Awesome CDN & Local SVGs/PNGs

## 💻 Local Setup & Development

To view and run this project locally, simply follow these steps:

1. Clone the repository to your local machine:
   ```bash
   git clone https://github.com/refat7321-51/Donation.git
   ```
2. Navigate to the project folder:
   ```bash
   cd Donation
   ```
3. Open `index.html` in your favorite web browser (double-click the file, or serve it using Live Server/dev server).
